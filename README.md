# 📰 News_api: 經濟日報各大類別新聞擷取與處理腳本

## ✨ 專案簡介
本專案是一個簡單、輕量級的 **Python 腳本**，物錢主要是爬取**經濟日報各大類別新聞**，**自動化地擷取最新的新聞標題與文章資料**。
---

## 🚀 主要功能

* **API 串接**：透過 Python 的 `requests` 函式庫請求經濟日報網頁的內容。
* **資料擷取**：依據配置的參數，取得最新的新聞標題、來源、發布時間和文章 URL 等關鍵資訊。
* **資料處理**：將 API 返回的 JSON 格式資料進行初步的解析與結構化處理。
